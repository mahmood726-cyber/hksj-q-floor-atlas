# hksj-q-floor-atlas

**Status:** Pre-registration in progress (spec under user review, no compute yet).

Quantifies the internal impact of the HKSJ Q-floor (`max(1, Q/(k−1))`) across the Pairwise70 Cochrane corpus (n = 6,386 MAs, REML τ²). Sister project to `cochrane-modern-re`, `repro-floor-atlas`, `responder-floor-atlas`.

See [`docs/spec.md`](docs/spec.md) for the full design spec.

## Headline question

Across the I²=0 subset of Pairwise70 (~3,500 MAs where `Q ≤ k−1` and HKSJ-no-floor narrows the SE below REML+Wald), what is the distribution of the precision overstatement and how often does applying the floor flip the significance verdict or directional conclusion?
